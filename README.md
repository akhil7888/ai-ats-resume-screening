# AI ATS Resume Screening — Gradio Version

## Features
✔ ATS Resume Score  
✔ ATS Keyword Analysis  
✔ Resume Improvement  
✔ Job Description Generator  
✔ Recruiter Feedback  
✔ Chat with Resume  

## Deployment (HuggingFace Spaces)
1. Create new Space → Gradio
2. Upload all files
3. Add Secret:
   - GROQ_API_KEY = your_key_here
4. Run

App will start automatically.
